##  Overview
**Dr. Stone** is a Japanese **manga** and **anime** by *Riichiro Inagaki* and *Boichi*.  
After humanity turns to stone for 3,700 years, **Senku Ishigami**, a genius scientist, aims to rebuild civilization using science.

---
